# TASK MANAGEMENT API APPLICATION

---

## Overview 

A simple task management application built during Express API course in order to gain a solid understanding and knowledge about API writing concepts. 

The goal is to provide a simple interface includes features that can help users manage their tasks logically and efficiently. 

This project focuses on a lightweight and practicial workflow, User Interface design might not be focused. React for Front-end developmen will be added to make this project a full-stack project.

---

## Features

- Browse upcomming tasks
- Add tasks
- Modify tasks
- Delete tasks
